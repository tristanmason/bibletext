# bibletext
Bibletext is a Wordpress plugin that displays ESV Bible text from a shortcode or URL query.

## Getting Started

### Prerequisites

Wordpress running on a web server.

### Installing

Download the plugin, zip the files, and upload .zip to Wordpress.
